### 2. 서버 초기 설정 완료

#### ✅ Node.js 서버 환경 구축
- **Express.js** - 웹 서버 프레임워크
- **Socket.io** - 실시간 통신
- **SQLite3** - 데이터베이스
- **CORS** - 크로스 오리진 지원
- **Body-parser** - 요청 데이터 파싱
- **Nodemon** - 개발용 자동 재시작

#### ✅ 서버 실행 확인
```bash

```
- 포트 8000에서 정상 실행
- 자동 재시작 기능 활성화
- 기본 API 엔드포인트 동작 확인

### 3. Python 클라이언트 초기 설정 완료

#### ✅ Python 가상환경 구축
- **가상환경 생성**: `venv/`
- **필수 패키지 설치**:

  - `websocket-client` - WebSocket 클라이언트
  - `pyinstaller` - 실행 파일 생성
